#### How-To

Install wrangler with npm:
```
npm install -g wrangler
```
Put your account_id in `wranger.toml` or run `wrangler login`

To test locally, run:
```
wrangler dev
```
or
```
wrangler preview --watch
```

To publish, run:
```
wrangler publish
```
